--------------------------lancment de l'application ---------------
1. Demarrer le serveur hdfs en tapant les commandes
 * hdfs namenode -format
 * start-dfs.cmd
 * start-yarn.cmd

2. Dans backend sur l'invite tapez la commande "python app.py" pour lancer le serveur flask

3. Dans frontend taper la commande "npm install" pour installer les dependances et ensuite 
taper "npm run dev" pour lancer l application

Nb: Assurer vous d'etre connecté à internet pour lancer l application
